Soal No 1

Menggunakan HTML, CSS serta JavaScript dalam pemrograman web?

  • HTML (HyperText Markup Language)

    • HTML adalah fondasi dari setiap halaman web. Fungsi utamanya adalah menyediakan struktur dasar dan konten, seperti judul, paragraf, gambar, dan tautan. HTML menentukan layout awal dan elemen-elemen yang akan ditampilkan kepada pengguna. Dengan HTML, kita dapat menambahkan elemen-elemen penting seperti formulir, tabel, atau daftar yang membantu menyampaikan informasi kepada pengguna.

  • CSS (Cascading Style Sheets)

    • CSS bertanggung jawab atas tampilan atau style dari halaman web. Dengan CSS, pengembang dapat mengontrol aspek visual, seperti warna, tata letak, ukuran font, dan posisi elemen. CSS membantu memisahkan tampilan visual dari konten (yang diatur oleh HTML), sehingga memungkinkan pengembang untuk mendesain halaman web yang lebih konsisten dan mudah dikelola. CSS juga mendukung responsive design, yaitu desain yang menyesuaikan dengan ukuran layar perangkat yang digunakan, sehingga halaman web tetap terlihat baik di desktop, tablet, maupun smartphone.

  • JavaScript

    • JavaScript adalah bahasa pemrograman yang membuat halaman web menjadi interaktif. Dengan JavaScript, kita dapat menambahkan fungsionalitas seperti tombol yang merespons klik pengguna, animasi, dan validasi formulir. JavaScript juga memungkinkan pengembangan single-page applications (SPA) dengan menggunakan kerangka kerja seperti React, Angular, atau Vue, sehingga interaksi antar halaman bisa lebih cepat tanpa harus memuat ulang seluruh halaman.

Soal No 2

HTML, CSS serta JavaScript dapat diintegrasikan dalam pemrograman web?

  • Penggunaan Langsung dalam File HTML

    • CSS di dalam HTML: CSS dapat diintegrasikan dalam file HTML menggunakan elemen style di dalam head.

    JavaScript di dalam HTML: JavaScript dapat ditambahkan langsung menggunakan elemen script, biasanya diletakkan di bagian bawah body untuk memastikan elemen HTML dimuat terlebih dahulu.

  • File Terpisah (Modularisasi)

    • Menggunakan file terpisah untuk HTML, CSS, dan JavaScript membuat kode lebih terorganisir dan mudah dikelola. Ini juga memungkinkan pemuatan ulang yang lebih cepat, karena file CSS dan JavaScript dapat di-cache oleh browser.

  • Manipulasi DOM dengan JavaScript

    • JavaScript memungkinkan pengembang untuk memodifikasi elemen HTML secara dinamis. Dengan memanipulasi DOM (Document Object Model), kita dapat menambahkan, mengubah, atau menghapus elemen pada halaman.
